== Description ==

Auto Login plugin is ideal plugin for login from url, it bypass wordpress login screen and login the users from url.

Basically it creates a virtual url for user, so selected roles users can login through url which contain user's username and redirect url and the salt which is a SHA256 hash string containing

user name and secret key which is randomly generate when plugin activated. so when user go to that url if user authenticated than it will redirect to Success url and if not authenticated than 

it will redirect to Error Url. Both Success and Error Url's and User Roles who are able to login from url are Set from Autologin Plugin Settings.


== Installation ==

Upload the Auto Login plugin to your site, Activate it, You're done!
